Brain Tumor Detection — Demo

This is a lightweight web UI deployed on Vercel. For predictions, it calls an external Python backend (set via the BACKEND_URL env var) through a proxy at /api/predict.


Tip: You'll set the BACKEND_URL in your Vercel project to point to your Python API hosted on Render/Railway/Hugging Face Spaces.

Built for Vercel hosting. Backend runs where Python + PyTorch are supported.
